✈️ Chatbot Travel Assistant

Chatbot asisten perjalanan pintar yang menggunakan **Google Gemini AI**. Membantu pengguna membuat itinerary liburan dengan tone yang ramah.

## 🛠️ Instalasi & Menjalankan

1. **Clone & Masuk ke Folder**
   ```bash
   git clone https://github.com/naylahafsah-itpln/Chatbot-travel-assistant.git
   cd gemini-chatbot-api`

1. **Install Library**Bash
    
    `npm install`
    
2. **Setting API Key**Code snippet
    
    Buat file `.env` dan isi dengan:
    
    `GEMINI_API_KEY=isi_api_key_kamu`
    
3. **Run Server**Bash
    
    # 
    
    `node index.js`
    
    Buka browser di: `http://localhost:3000`
